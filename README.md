# Weather App

A simple and responsive web application that provides current weather information for any city of your choice. The Weather App is built using JavaScript, HTML, and CSS.

## Features

- Search for the current weather by city name
- Responsive design for mobile and desktop devices
- Clean and intuitive user interface
- Real-time weather data (API integration required)

## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/Bryannnnn-1/Weather-App.git
    cd Weather-App
    ```

2. **(Optional) Get a Weather API Key**
    - Sign up at [OpenWeatherMap](https://openweathermap.org/api) or another weather API provider
    - Obtain an API key for fetching weather data

3. **Configure the API key**
    - Open the JavaScript source file (e.g., `app.js` or similar)
    - Replace the placeholder with your API key:
      ```js
      const apiKey = "YOUR_API_KEY_HERE";
      ```

4. **Open the App**
    - Open `index.html` in your browser
