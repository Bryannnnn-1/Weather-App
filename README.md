# Weather App

A simple and responsive web application that provides current weather information for any city of your choice. The Weather App is built using JavaScript, HTML, and CSS (Bootstrap).

## Features

- Search for the current weather by city name
- Responsive design for mobile and desktop devices
- Clean and intuitive user interface
- Real-time weather data (API integration required)

## Technologies Used

- **JavaScript**: Handles the fetching of weather data and application logic
- **HTML**: Structures the content of the web application
- **CSS/Bootstrap**: Styles the app and ensures responsiveness
